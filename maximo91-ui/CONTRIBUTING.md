# 贡献指南

欢迎为 Maximo Helper Box UI 项目做出贡献！

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请通过 GitHub Issues 提交。

### 提交代码

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

### 开发设置

```bash
# 克隆仓库
git clone https://github.com/your-username/maximo-helper-box-ui.git

# 进入目录
cd maximo-helper-box-ui

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 代码规范

- 遵循现有的代码风格
- 确保代码通过所有测试
- 为新功能添加适当的文档
- 更新 README.md（如果需要）

### 提交信息规范

请使用清晰的提交信息，说明您所做的更改：

- `feat:` 新功能
- `fix:` 修复 bug
- `docs:` 文档更新
- `style:` 代码格式更改
- `refactor:` 代码重构
- `test:` 添加或修改测试
- `chore:` 构建过程或辅助工具的变动

## 行为准则

请尊重其他贡献者，保持友好和专业的交流氛围。

## 许可证

通过贡献代码，您同意您的贡献将在 MIT 许可证下发布。
