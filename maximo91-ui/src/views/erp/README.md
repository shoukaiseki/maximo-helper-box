# 公开ERP示例页面

本目录包含了一些公开的ERP系统示例页面，用于展示如何使用 Maximo Vue 组件库构建企业级应用界面。

## 页面列表

### 列表页面

#### 1. 采购订单列表 (PurchaseOrderList.vue)
- **路由**: `/purchase-order-list`
- **功能**: 展示采购订单的列表信息
- **字段**: 订单号、供应商、订单日期、交货日期、总金额、状态、创建人

#### 2. 供应商管理列表 (SupplierList.vue)
- **路由**: `/supplier-list`
- **功能**: 展示供应商的基本信息
- **字段**: 供应商编码、供应商名称、联系人、联系电话、邮箱、评级、状态

#### 3. 员工管理列表 (EmployeeList.vue)
- **路由**: `/employee-list`
- **功能**: 展示公司员工的基本信息
- **字段**: 员工编号、姓名、部门、职位、电话、邮箱、入职日期、状态

#### 3.5. 员工详情 (EmployeeDetail.vue) ⭐ 综合示例
- **路由**: `/employee-detail`
- **功能**: 员工详细信息，**使用了所有组件**的综合示例页面
- **特点**:
  - ✨ **MaximoButtonGroup**: 顶部按钮组（打印、导出、分享）
  - ✨ **MaximoMultipartTextbox**: 多部分文本框（地址分为省/市、区/县、街道、邮编）
  - ✨ **MaximoTableDetail**: 表格详情组件
  - ☑️ **MaximoCheckbox**: 复选框（社保、公积金）
  - 📝 **MaximoMultilineTextbox**: 多行文本框（备注信息）
  - 📎 **MaximoAttachments**: 附件列表（身份证、学历证、简历等）
  - 📑 **4个标签页**: 基本信息、培训记录、绩效考核、奖惩记录
  - 📊 **3个子表**: 培训记录、绩效考核、奖惩记录

#### 4. 设备维护工单列表 (WorkOrderList.vue)
- **路由**: `/work-order-list`
- **功能**: 展示设备维护工单信息
- **字段**: 工单号、设备名称、设备编号、故障类型、优先级、报修日期、报修人、状态

#### 5. 资产管理列表 (AssetList.vue)
- **路由**: `/asset-list`
- **功能**: 展示公司资产信息
- **字段**: 资产编号、资产名称、资产类别、规格型号、购置日期、原值、净值、使用部门、状态

### 详情页面（多标签页 + 多子表）

#### 6. 采购订单详情 (PurchaseOrderDetail.vue)
- **路由**: `/purchase-order-detail`
- **功能**: 采购订单详细信息，包含3个子表
- **特点**:
  - 使用 MaximoSection、MaximoSectionRow、MaximoSectionCol 布局
  - 使用 MaximoTabs 和 MaximoTab 实现多标签页
  - 子表1：订单明细（物料清单）
  - 子表2：付款记录（预付款、进度款、尾款）
  - 子表3：交货记录（分批交货信息）

#### 7. 供应商详情 (SupplierDetail.vue)
- **路由**: `/supplier-detail`
- **功能**: 供应商详细信息，包含3个子表和附件
- **特点**:
  - 使用 MaximoTextbox、MaximoMultilineTextbox、MaximoCheckbox
  - 使用 MaximoAttachments 展示资质文件
  - 子表1：供应产品列表
  - 子表2：评估记录（质量、交货、服务评分）
  - 子表3：合作订单历史

#### 8. 设备维护工单详情 (WorkOrderDetail.vue)
- **路由**: `/work-order-detail`
- **功能**: 工单详细信息，包含3个子表和附件
- **特点**:
  - 使用 MaximoMultilineTextbox 展示故障描述
  - 使用 MaximoAttachments 展示故障照片和维修报告
  - 子表1：维修记录（维修过程）
  - 子表2：备件使用（更换的配件）
  - 子表3：费用明细（备件费、人工费、其他）

#### 9. 资产详情 (AssetDetail.vue)
- **路由**: `/asset-detail`
- **功能**: 资产详细信息，包含3个子表和附件
- **特点**:
  - 使用 MaximoCheckbox 展示投保和维护标志
  - 使用 MaximoAttachments 展示设备文档
  - 子表1：折旧记录（月度折旧明细）
  - 子表2：维修历史（保养和维修记录）
  - 子表3：使用记录（日常使用情况）

## 使用的组件

✅ **所有 `src/components` 目录下的15个组件都已在这些页面中使用！**

### 布局组件
- ✅ **MaximoSection**: 区块组件，用于组织页面内容
- ✅ **MaximoSectionRow**: 行布局组件
- ✅ **MaximoSectionCol**: 列布局组件，支持 :span 属性（8/12/16/24）

### 表单组件
- ✅ **MaximoTextbox**: 文本框组件，支持 readonly 属性
- ✅ **MaximoMultilineTextbox**: 多行文本框，支持 :rows 属性
- ✅ **MaximoMultipartTextbox**: 多部分文本框，将多个字段组合在一起（如地址）
- ✅ **MaximoCheckbox**: 复选框组件，支持 disabled 属性
- ✅ **MaximoLabel**: 标签组件

### 数据展示组件
- ✅ **MaximoTable**: 表格组件，用于展示数据列表
- ✅ **MaximoTableDetail**: 表格详情组件，用于在标签页中展示子表数据
- ✅ **MaximoAttachments**: 附件组件，展示文件列表

### 导航组件
- ✅ **MaximoTabs**: 标签页容器
- ✅ **MaximoTab**: 单个标签页

### 操作组件
- ✅ **MaximoButton**: 按钮组件
- ✅ **MaximoButtonGroup**: 按钮组组件，一组相关按钮

## 特点

1. **统一的样式**: 所有页面都遵循 Maximo ERP 系统的设计风格
2. **响应式布局**: 页面适配不同屏幕尺寸
3. **交互功能**: 支持行点击、工具栏操作等交互
4. **模拟数据**: 每个页面都包含了示例数据，方便演示和测试

## 使用方法

这些页面已经配置好路由，可以直接在应用中访问。启动开发服务器后，可以通过左侧菜单导航到各个页面。

```bash
npm run dev
```

然后在浏览器中访问对应的路由即可。
