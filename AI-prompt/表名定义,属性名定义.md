# 一、表名命名规则
1. 新建业务表：以 IBM_ 开头，英文大写，无下划线，简洁缩写，长度≤30字符
   例：IBM_SUPPINVITE、IBM_BID、IBM_BIDLINE、IBM_ST、IBM_BIDNOTICE
2. 系统已有表：沿用原名，不加前缀，长度≤30字符
   例：COMPANIES、RFQ、RFQTERM、RFQSTATUS、QUOTATIONLINE
3. 明细表：主表名 + LINE，长度≤30字符
   例：IBM_BIDLINE、IBM_STLINE

   # 二、字段名命名规则
1. 字段统一：英文大写，无下划线，简洁缩写，长度≤50字符
2. 编号主键：XXXNUM
   例：SINUM、BIDNUM、STNUM、BNNUM
3. 外键字段：直接引用对方主键名
   例：RFQNUM、BIDNUM、STNUM
4. 标准字段：
   - 描述：DESCRIPTION
   - 创建人：CREATEBY、CREATOR
   - 修改人：CHANGEBY
   - 创建日期：CREATEDATE
   - 修改日期：CHANGEDATE
   - 状态：STATUS
5. 已有表新增字段：以 IBM_ 开头，英文大写，长度≤50字符
   例：IBM_SINUM
